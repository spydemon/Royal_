Collection information
======================

Cette fenêtre permet de sélectionner la collection du nouvel album. Si cette collection n'existe pas, elle pourra être créée à l'aide de cette fenêtre.

$wizard.img$

Name
....

* The name of the Collection.
* This field is compulsory.

Editor
.......

* The name of the editor relative to the collection
* You have to choose it among those listed in the database. if the editor does not exist, you can create it by entering it's name

Web
...

* The web site of the collection


Description
...........

* Description of the collection.
